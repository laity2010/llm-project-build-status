## SEQUENCES

### 序列1

#### A
光子单个能量极低，可见光光子只有约2-3 eV，就算太阳光每秒有10¹⁷个光子打到你1cm²脸上，总功率也才0.1瓦左右，热量积累很慢，完全不足以引发任何爆炸。
蘑菇云需要瞬间释放相当于几万吨TNT的能量（10¹³焦耳以上），而光子打脸的能量密度差了十几个数量级，最多让你觉得暖和或晒黑，离核爆级蘑菇云差着宇宙鸿沟。

#### B
简单来说，是因为光子的动量极小且能量等级不足以触发核反应。虽然根据质能方程 E=mc
2
 能量与质量存在等价关系，但光子没有静止质量，单个可见光子的能量极其微弱。要产生“蘑菇云”这种核爆炸效果，需要极高密度的能量在极短时间内诱发原子核层面的连锁反应（如裂变或聚变）。

阳光照在脸上时，光子只是被皮肤分子吸收并转化为热能，让分子动得快一点点而已。如果每个光子都能撞出蘑菇云，那么人类在拉开窗帘的一瞬间就会遭遇无数次“核平”。

#### C
要出现“蘑菇云”，必须满足一个核心条件：在极短时间内把巨大的能量释放到很小的空间里，从而产生高温火球和强烈冲击波。光子撞到脸上完全不满足这个条件。

第一，单个光子的能量非常小。
光子的能量由公式 E = hν 决定（h 为普朗克常数）。可见光光子的能量大约在 10⁻¹⁹ 焦耳量级，这是一个极其微小的能量。
当光子撞到皮肤时，它通常只是被反射，或者被皮肤中的电子吸收，然后转化为一点点热量，让分子振动稍微增加。

第二，蘑菇云需要爆炸级别的能量。
无论是核爆还是大型化学爆炸，本质都是在极短时间内释放巨大的能量。典型规模大约在 10¹²–10¹⁵ 焦耳。这样的能量会把空气瞬间加热到极高温度，形成火球并产生强烈冲击波。热空气随后快速上升并卷起周围空气和尘土，才形成蘑菇状云。

第三，光子照射是分散而缓慢的能量输入。
即使是正午阳光，地面接收到的功率也只有大约 1000 W/m²。照到脸上的面积可能只有 0.02 m² 左右，也就是几十瓦的功率，相当于每秒几十焦耳的能量，而且是持续、均匀地输入。

第四，物理过程完全不同。
光子撞到脸上时发生的是：
光子 → 被吸收或反射 → 能量变成微弱热量 → 在皮肤和空气中迅速扩散。

而爆炸则是：
巨量能量瞬间释放 → 高温火球 → 冲击波 → 强烈对流 → 蘑菇云。

只有在极端情况下，例如把极大量光子在极短时间、极小区域内集中（比如超高功率激光），才可能产生等离子体和冲击波，出现类似微型爆炸的现象。

总结：光子撞到脸上不会产生蘑菇云，因为能量太小、能量密度太低，而且释放方式是分散而缓慢的，只会带来一点点热。
