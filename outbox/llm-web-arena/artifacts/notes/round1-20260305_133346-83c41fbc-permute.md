## SEQUENCES

### 序列1

#### A
光子单个能量极低（可见光一个光子只有几eV），就算亿亿个光子同时打到脸上，总能量也才零点几焦耳，远远不够引发核聚变或任何爆炸级能量释放。
蘑菇云需要极高功率密度和极大量能量瞬间沉积（如核弹当量级），而阳光即使聚焦到极致也只是热效应，最多让你脸上起个水泡，离蘑菇云差了十几个数量级。

#### B
简单来说，是因为光子的质量（动量）极小且能量等级太低。虽然爱因斯坦的方程 E=mc
2
 揭示了质量与能量的转换关系，但光子没有静止质量，单个可见光子的能量仅够让电子在原子能级间跳跃，或者让你的皮肤分子振动得快一点（产生热感）。要产生“蘑菇云”这种核爆炸效果，需要极高密度的能量在极短时间内诱发原子核层面的连锁反应。如果每个光子都能撞出蘑菇云，那么每天清晨你拉开窗帘的那一刻，就不是迎接阳光，而是迎接无数场微型核爆了。

#### C
因为蘑菇云需要的是“极其巨大的能量在极短时间、极小体积里集中释放”，而普通光子既没有这么大的能量，也没有这么高的能量密度。

1. 单个光子的能量非常小
光子的能量由公式 E = hν 决定。可见光光子的能量大约只有 10⁻¹⁹ 焦耳量级。
当光子撞到脸上时，它通常只是被反射，或者被皮肤中的电子吸收，然后转化为一点点热量。

2. 蘑菇云来自爆炸
蘑菇云是爆炸产生的对流结构。爆炸会在极短时间内释放巨大的能量（例如核爆约 10¹³–10¹⁵ 焦耳），把空气瞬间加热成高温火球，并产生强烈冲击波。随后热空气迅速上升、卷起周围空气和尘土，就形成了蘑菇状云。

3. 光子照射是“缓慢、分散”的能量输入
即使是正午阳光，地面能量通量也只有大约 1000 W/m²。照到脸上的功率大约只有几十瓦，也就是每秒几十焦耳，而且是持续、均匀地释放，不会产生冲击波或火球。

4. 只有在极端情况下光子才可能造成类似爆炸
如果把极大量光子在极短时间和极小空间内集中，比如超高功率激光照射材料，就可能产生等离子体、冲击波甚至微型爆炸。但那是因为能量密度已经达到了爆炸级别。

总结：
光子撞脸时的能量太小、太分散，只会变成一点点热；而蘑菇云需要的是瞬间释放的巨大能量和强烈冲击波，所以日常光子照到脸上不会出现蘑菇云。
